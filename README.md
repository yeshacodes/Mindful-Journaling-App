# Mindful Micro-Journaling App

A calm, minimal journaling application built with Next.js, TypeScript, and Tailwind CSS.

## Project Structure

- **next-app/**: The main application.
- **legacy_src_backup/**: Backup of the previous React/Vite implementation (kept for reference).

## Getting Started

1. Navigate to the app directory:
   ```bash
   cd next-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser.
